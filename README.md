Интернет-магазин STORE
Профессиональный интернет-магазин на Spring Boot
Функциональные возможности
Для покупателей
📱Адаптивный дизайн — работает на всех устройствах
🔍 Умный поиск — по названию и описанию товаров
🗂️ Категории — фильтрация по разделам (Электроника, Компьютеры, Книги и др.)
🛒 Корзина — AJAX-добавление без перезагрузки страницы
💳 Оформление заказа — форма с валидацией данных
👁️ Изображения товаров — загрузка и отображение реальных фото
Для администратора
👔 Админка товаров — CRUD-операции через веб-интерфейс
📦 Админка заказов — управление статусами (Новый → В обработке → Доставлен)
📊 Просмотр всех заказов — детальная информация по каждому заказу
Технические особенности
🗄️ PostgreSQL — production-ready база данных
🖼️ Загрузка изображений — через веб-интерфейс
🔐 Безопасность — защита от XSS и SQL-инъекций
📱 Mobile-first — современный адаптивный дизайн
